# Nested Lists

Lists can be nested by indenting with spaces.

## Nested Unordered

- Parent item
  - Child item
  - Another child
    - Grandchild item
    - Another grandchild
  - Back to child level
- Back to parent level

## Nested Ordered

1. First parent
   1. First child
   2. Second child
      1. First grandchild
      2. Second grandchild
2. Second parent
   1. Child of second

## Mixed Nesting

- Unordered parent
  1. Ordered child
  2. Another ordered
     - Unordered grandchild
     - Another unordered
- Back to unordered parent
  1. More ordered children

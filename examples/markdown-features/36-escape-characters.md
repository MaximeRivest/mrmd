# Escape Characters

Backslash escapes Markdown special characters.

## Asterisks

\*This is not italic\*

\*\*This is not bold\*\*

## Underscores

\_This is not italic\_

\_\_This is not bold\_\_

## Backticks

\`This is not code\`

## Hashes

\# This is not a heading

\#\#\# This is not H3

## Brackets and Parentheses

\[This is not a link\](https://example.com)

\!\[This is not an image\](image.png)

## Other Characters

\- Not a list item

\1. Not an ordered list

\> Not a blockquote

\| Not a table \|

## All Escapable Characters

\\   Backslash
\`   Backtick
\*   Asterisk
\_   Underscore
\{\} Curly braces
\[\] Square brackets
\(\) Parentheses
\#   Hash
\+   Plus
\-   Minus
\.   Dot
\!   Exclamation
\|   Pipe

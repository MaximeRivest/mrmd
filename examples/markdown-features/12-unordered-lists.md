# Unordered Lists

Unordered lists can use hyphens, asterisks, or plus signs.

## Using Hyphens

- Item one
- Item two
- Item three
- Item four

## Using Asterisks

* First item
* Second item
* Third item

## Using Plus Signs

+ Apple
+ Banana
+ Cherry

## Mixed (Not Recommended)

- Hyphen item
* Asterisk item
+ Plus item

# Inline Links

Links are created with square brackets for text and parentheses for URL.

## Basic Link

[Visit GitHub](https://github.com)

## Link with Title

[Link with title](https://github.com "GitHub Homepage")

Hover over the link above to see the title.

## Multiple Links

Check out [Google](https://google.com), [GitHub](https://github.com), and [Stack Overflow](https://stackoverflow.com).

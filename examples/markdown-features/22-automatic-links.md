# Automatic Links

URLs and emails can be auto-linked with angle brackets.

## URLs

<https://www.example.com>

<https://github.com>

<http://localhost:3000>

## Email Addresses

<email@example.com>

<contact@company.org>

<support@website.net>

## Without Angle Brackets (Extended Syntax)

Some parsers auto-link bare URLs:

https://www.example.com

www.example.com

## Comparison

With brackets: <https://github.com>

Bare URL: https://github.com

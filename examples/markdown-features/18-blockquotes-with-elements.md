# Blockquotes with Other Elements

Blockquotes can contain other Markdown elements.

## With Heading

> ### This is a heading inside a quote
>
> And this is the content below the heading.

## With List

> Here's a quote with a list:
>
> - First item
> - Second item
> - Third item

## With Formatting

> **Bold text** and *italic text* work in quotes.
>
> You can also use `inline code` and ~~strikethrough~~.

## With Code Block

> Here's some code:
>
> ```javascript
> console.log("Hello from a quote!");
> ```

## Combined

> ### Quote Title
>
> This quote has multiple elements:
>
> - **Bold item**
> - *Italic item*
> - `Code item`
>
> And a final paragraph with a [link](https://example.com).

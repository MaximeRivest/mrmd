# Subscript

Subscript text appears below the baseline.

## Chemical Formulas

H~2~O is water.

CO~2~ is carbon dioxide.

C~6~H~12~O~6~ is glucose.

H~2~SO~4~ is sulfuric acid.

## Mathematical Notation

X~0~, X~1~, X~2~, ... X~n~

a~i~ + b~i~ = c~i~

log~2~(8) = 3

## Variable Indices

The sequence A~1~, A~2~, A~3~ continues.

Element E~ij~ is in row i, column j.

## HTML Alternative

Using HTML: H<sub>2</sub>O

CO<sub>2</sub> with subscript tag.

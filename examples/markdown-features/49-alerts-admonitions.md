# Alerts / Admonitions

GitHub-style callouts for important information.

## Note

> [!NOTE]
> This is a note callout. Use it for additional information that readers might find helpful.

## Tip

> [!TIP]
> This is a tip callout. Use it for helpful advice or shortcuts.

## Important

> [!IMPORTANT]
> This is important information that users need to know.

## Warning

> [!WARNING]
> This is a warning callout. Use it to alert users about potential issues.

## Caution

> [!CAUTION]
> This is a caution callout. Use it for dangerous operations or irreversible actions.

---

## With Additional Content

> [!NOTE]
> Notes can contain multiple paragraphs.
>
> Like this second paragraph.
>
> - And lists
> - With items

> [!WARNING]
> Be careful when running:
> ```bash
> rm -rf /
> ```
> This command is destructive!

# Bold and Italic Combined

Text can be both bold and italic simultaneously.

## Triple Asterisks

This is ***bold and italic*** text.

## Triple Underscores

This is ___bold and italic___ text.

## Mixed Syntax

This is **_bold and italic_** using mixed syntax.

This is *__bold and italic__* another way.

This is _**also bold and italic**_ mixing differently.

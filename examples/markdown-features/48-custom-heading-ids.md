# Custom Heading IDs

Override automatic heading IDs with custom ones.

## Default ID Generation

### This Heading Has Spaces {#custom-spaces}

The default ID would be `this-heading-has-spaces`, but we set it to `custom-spaces`.

### Technical Details! {#tech-details}

Special characters are normally stripped, but we can use a clean custom ID.

### 2024 Updates {#updates-2024}

Numbers at the start can be problematic, so custom IDs help.

# Raw URLs

URLs can be auto-linked without special syntax.

## With Angle Brackets (Standard)

<https://github.com>

<https://www.example.com>

<http://localhost:3000>

## Bare URLs (Extended)

Many parsers auto-link bare URLs:

https://github.com

https://www.google.com

https://stackoverflow.com/questions

## WWW URLs

www.example.com

www.github.com

## File URLs

file:///home/user/document.txt

## FTP URLs

ftp://ftp.example.com/files

## In Text

Visit https://github.com for more information.

Check out www.example.com for examples.

## Comparison

Angle brackets: <https://example.com>

Bare URL: https://example.com

Inline link: [Example](https://example.com)

Reference link: [Example][ref]

[ref]: https://example.com

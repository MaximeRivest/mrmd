# Headings

Markdown supports 6 levels of headings using `#` symbols.

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Alternative syntax for H1 and H2:

Heading 1
=========

Heading 2
------------



## More Heading Examples

### Nested Structure
You can create hierarchical document structures:

# Main Title
## Section 1
### Subsection 1.1
#### Detail 1.1.1
### Subsection 1.2
## Section 2
### Subsection 2.1

### Headings with Formatting
Headings can include **bold**, *italic*, and `code`:

## This is a **Bold** Heading
### This heading has *emphasis*
#### Heading with `code` inline

### Best Practices
- Use only one H1 per document
- Don't skip heading levels
- Keep headings concise and descriptive
- Use sentence case or title case consistently

# Footnotes

Footnotes add references that appear at the bottom.

## Basic Footnotes

Here's a sentence with a footnote.[^1]

Another sentence with a different footnote.[^2]

[^1]: This is the first footnote.
[^2]: This is the second footnote.

## Named Footnotes

This is an important statement.[^note]

Here's another claim that needs citation.[^source]

[^note]: This is a named footnote explaining the statement.
[^source]: Source: Example Study, 2024.

## Long Footnotes

Here's a sentence with a longer footnote.[^long]

[^long]: This is a longer footnote that contains
    multiple lines and paragraphs.

    You can include code:

        def example():
            return True

    And other elements.

## Inline Footnotes (Some Parsers)

This is a test^[This is an inline footnote] in the text.

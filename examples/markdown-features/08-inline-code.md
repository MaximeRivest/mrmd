# Inline Code

Inline code uses single backticks.

Use the `print()` function to output text.

Reference a `variable` or `filename.js` inline.

The `const` keyword declares a constant.

Run `npm install` to install dependencies.

## Backticks Inside Code

Use double backticks to include a literal backtick: `` `code` ``

Or: ``There's a `backtick` inside``

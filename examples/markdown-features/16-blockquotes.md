# Blockquotes

Blockquotes use the greater-than symbol.

## Single Line

> This is a blockquote.

## Multiple Lines

> This is a blockquote that spans
> multiple lines. Each line starts
> with a greater-than symbol.

## Lazy Wrapping

> This is a blockquote that spans multiple lines
but only the first line needs the symbol.
This is sometimes called lazy wrapping.

## Multiple Paragraphs

> First paragraph of the quote.
>
> Second paragraph of the quote.
>
> Third paragraph with a blank quoted line between.

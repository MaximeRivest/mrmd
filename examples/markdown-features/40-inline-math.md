# Inline Math (LaTeX)

Inline math uses single dollar signs.

## Basic Formulas

The equation $E = mc^2$ is famous.

The Pythagorean theorem: $a^2 + b^2 = c^2$

## Variables

If $x = 5$ and $y = 3$, then $x + y = 8$.

## Fractions

The fraction $\frac{1}{2}$ equals 0.5.

Calculate $\frac{a + b}{c - d}$ to find the result.

## Greek Letters

The value of $\pi$ is approximately 3.14159.

Use $\alpha$, $\beta$, $\gamma$ for angles.

The sum $\Sigma$ and product $\Pi$ operators.

## Subscripts and Superscripts

Variables: $x_1$, $x_2$, $x_n$

Powers: $2^{10}$, $e^{i\pi}$

Combined: $a_i^2 + b_i^2 = c_i^2$

## Common Operations

Square root: $\sqrt{16} = 4$

nth root: $\sqrt[3]{8} = 2$

Absolute value: $|x - y|$

# Task Lists (Checkboxes)

Task lists use brackets with a space or x for status.

## Basic Task List

- [x] Completed task
- [x] Another completed task
- [ ] Incomplete task
- [ ] Another todo item

## Project Checklist

- [x] Set up project structure
- [x] Install dependencies
- [x] Write initial code
- [ ] Add unit tests
- [ ] Write documentation
- [ ] Deploy to production

## Nested Tasks

- [ ] Main task
  - [x] Subtask completed
  - [ ] Subtask pending
  - [ ] Another subtask
- [x] Another main task
  - [x] All subtasks done
  - [x] This one too

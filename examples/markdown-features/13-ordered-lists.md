# Ordered Lists

Ordered lists use numbers followed by periods.

## Standard Numbering

1. First item
2. Second item
3. Third item
4. Fourth item

## Auto-Numbering

Numbers don't have to be sequential - Markdown auto-numbers:

1. First item
1. Second item
1. Third item
1. Fourth item

## Starting from Different Number

5. This starts at five
6. Continues to six
7. And seven

## With Parentheses (Some Parsers)

1) First item
2) Second item
3) Third item

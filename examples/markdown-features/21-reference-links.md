# Reference Style Links

Reference links separate the link definition from its use.

## Basic Reference Links

Here is a [reference link][1] and [another one][2].

[1]: https://example.com
[2]: https://github.com

## Named References

Check out [GitHub][github] and [Google][google] for more info.

[github]: https://github.com
[google]: https://google.com

## With Titles

Visit [Example Site][example] for a demo.

[example]: https://example.com "Example Website"

## Implicit Link Names

[Google][] is a search engine.

[Google]: https://google.com

## Multiple Uses of Same Reference

First link to [GitHub][gh].
Second link to [GitHub][gh].
Third link to [GitHub][gh].

[gh]: https://github.com "GitHub"
